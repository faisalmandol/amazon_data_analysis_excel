📊 Amazon Sales Trend Analysis Dashboard | Microsoft Excel Project
Excited to share my latest Amazon Sales Trend Analysis Dashboard built entirely in Microsoft Excel.
This interactive dashboard transforms raw sales data into meaningful business insights through dynamic visualizations, KPIs, and slicers.

🔹 Key Highlights:
✅ Total Orders: 1,000
✅ Total Quantity Sold: 5,510
✅ Total Revenue: 322.97K
✅ Average Customer Rating: 6.97

📈 The dashboard provides insights into:
• Product line performance (Revenue vs Quantity Sold)
• Branch-wise revenue contribution
• City-wise sales analysis
• Payment method distribution
• Interactive filtering by date, city, and payment type

🛠️ Tools & Features Used:
• Microsoft Excel
• Pivot Tables
• Pivot Charts
• Slicers & Timelines
• Dashboard Design & Data Visualization

This project enhanced my understanding of data analysis, KPI tracking, and dashboard development while demonstrating how Excel can be used as a powerful business intelligence tool.
